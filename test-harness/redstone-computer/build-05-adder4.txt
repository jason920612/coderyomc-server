#   the ONLY thing that powers a dust from above (a strongly-powered solid block above a
#   dust does NOT power it — verified live). So a horizontal Cout(k)->Cin(k+1) wire cannot
#   reach the Cin ports without crossing tile internals. Tiling this top-fed tile gives 4
#   independent, difftest-clean full adders; a true ripple needs a full-adder variant with
#   a SIDE-ENTRY Cin bus (one clean port fanned internally), which is the next tile design.
